# Product Overview

Terminal Adventure Game is a browser-based text adventure game that provides an authentic terminal experience using xterm.js. Players navigate through interactive story-driven adventures using command-line style inputs.

## Core Features

- **Player Mode**: Navigate locations, interact with characters, and explore adventures
- **Admin Mode**: Create and manage custom adventures with sudo-style authentication (default password: `admin123`)
- **Persistent State**: Game progress is automatically saved via backend API
- **Demo Adventure**: Ships with "The Lost Temple" demo adventure

## User Experience

The game mimics a Unix-like terminal interface with:
- Command-line style prompts (`$` for player mode, `#` for admin mode)
- Familiar commands (move, look, talk, help, clear, exit)
- Color-coded output for different message types
- Loading indicators for async operations
